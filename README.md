# mac-provisioning
Provisioning files for Mac.

# Usage
## At the 1st time
```
$ curl -fsL https://raw.githubusercontent.com/kagemiku/mac-provisioning/master/install.sh > sh
```

## After the 2nd time
```
$ cd /path/to/this/repository
$ cd ansible
$ ansible-playbook -i hosts localhost.yml
```
